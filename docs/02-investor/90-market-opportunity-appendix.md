# Market Opportunity

## 1. Market thesis

LingoBites targets a large, practical learning behavior:

```text
Vietnamese learners encounter English in daily life but existing tools mostly translate, not teach.
```

The product opportunity is not to replace full English courses. The wedge is smaller and more concrete:

```text
Learn from real-world English the user already cares about.
```

---

## 2. Initial wedge

| Wedge | Why it matters |
|---|---|
| Vietnamese beginners | Need Vietnamese explanation and simple grammar |
| Real-world text | Higher relevance than generic textbook examples |
| Mobile scan/paste | Fits daily use and quick learning moments |
| Structured mini lesson | Turns a one-time translation into reusable learning material |

---

## 3. User situations

| Situation | Example |
|---|---|
| Student | Reads a short English paragraph or assignment |
| Office worker | Receives English email/message |
| Traveler | Reads sign/menu/flyer |
| Self-learner | Saves useful sentences for later |
| Parent/teacher support | Explains simple English to learner |

---

## 4. Why this can expand

Phase 0 starts with scan/paste → lesson. If users save and review lessons, the product can expand into a retention loop:

| Expansion | Business value |
|---|---|
| Flashcards | Higher repeat usage |
| Daily review | Habit formation |
| Speaking practice | Premium learning feature |
| AI tutor Q&A | Higher perceived value |
| PDF/web import | More input sources |
| Learning path | Subscription potential |

---

## 5. Assumptions to validate

This document intentionally avoids unsupported market-size claims. The first investor-grade proof should come from usage data.

| Assumption | Validation signal |
|---|---|
| Users want to learn from real-world English | Scan/paste usage frequency |
| Structured lesson is more valuable than translation | Qualitative feedback, save rate |
| Users return to review saved lessons | Review/open rate, D7 retention |
| Users will pay for managed AI convenience | Paid intent, trial-to-paid conversion |
| BYOK can support free/power users | BYOK setup completion rate |

---

## 6. Investor framing

The strongest story after beta would be:

```text
Users do not just translate. They generate lessons, save them, and return to review them.
```

That proves the product is moving from utility to learning habit.
