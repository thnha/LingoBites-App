# Competitive Landscape

## 1. Positioning map

LingoBites sits between translation, OCR, AI tutoring and traditional language learning.

```text
Generic translation → Understand meaning
Language learning app → Follow curriculum
LingoBites → Learn from real-world text
```

---

## 2. Comparison

| Product type | Examples | Strength | Gap |
|---|---|---|---|
| Translation | Google Translate, DeepL | Fast translation | Does not create structured beginner lesson |
| OCR/search | Google Lens | Strong image recognition | Not focused on learning loop |
| Curriculum apps | Duolingo | Habit and gamified curriculum | Not personalized to user-selected real-world text |
| Speaking apps | ELSA | Pronunciation training | Not focused on OCR/text-to-lesson |
| General AI chat | ChatGPT | Flexible explanations | No dedicated mobile scan/review/save UX |
| Dictionaries | Cambridge, Oxford, local dictionary apps | Word detail | Weak sentence-level contextual learning |

---

## 3. Differentiation

| Differentiator | Why it matters |
|---|---|
| Vietnamese beginner-first explanation | Targeted to user comprehension level |
| Real-world input | User learns from content they actually encounter |
| OCR review before AI | Reduces bad AI output from OCR mistakes |
| Structured lesson output | Enables consistent UI and saved lessons |
| Save/review loop | Turns one-time translation into learning asset |
| BYOK + managed paid mode | Controls AI cost while preserving paid UX |

---

## 4. Risk of competition

Large platforms can add similar features. The defensible angle is not OCR or AI alone, but the full beginner learning workflow:

- Vietnamese-specific explanation style
- beginner-friendly lesson structure
- local saved lesson/review loop
- quality gates around AI output
- cost-controlled AI access model
- focused UX for scan-to-learn, not generic chat

---

## 5. Strategic wedge

Start narrow:

```text
Vietnamese beginners learning from daily English.
```

Then expand:

- review/flashcards
- speaking practice from scanned sentences
- AI tutor Q&A per lesson
- PDF/web/article import
- personalized learning path
